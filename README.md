# 🧠 AI Exam Paper Generator

An AI-powered web application that generates structured exam papers from academic documents using Retrieval-Augmented Generation (RAG).

## 🚀 Features

* 📄 Upload PDF / Notes for context-based question generation
* 🧠 RAG-based system to reduce hallucination and improve accuracy
* ✏️ Generate MCQs, Short, and Long questions
* 🎯 Difficulty control (Easy, Medium, Hard)
* 🏷️ Cognitive tagging (Remember, Apply, etc.)
* 📥 Export exam papers as PDF and DOCX
* 🏫 Custom institutional branding support
* 🎨 Modern glassmorphism UI (dark theme)

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: FastAPI (Python)
* AI: Google Gemini API
* Document Processing: OCR + Text Extraction
* Architecture: RAG (Retrieval-Augmented Generation)
